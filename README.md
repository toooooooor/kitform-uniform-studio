# KITFORM — 3D Uniform Studio

白を基調にした、レスポンシブ対応のサッカーユニフォーム3Dカスタマイザーです。

## 起動

`index.html` をWebサーバーから配信してください。例:

```powershell
python -m http.server 4173 -d outputs/uniform-studio
```

ブラウザで `http://localhost:4173` を開きます。

Three.js とユーザー提供の `assets/uniform.glb` をプロジェクト内に同梱しているため、オフラインでも3D表示できます。

## 主な機能

- 18種類の柄と柄スケール変更
- 3色のリアルタイム配色
- 背番号・選手名・フォント・番号スタイル・個別位置調整
- 最大6個のロゴを番号で選択し、前後面・位置・サイズ・回転を個別調整
- マウス / タッチの上下左右回転、ズーム、正面 / 背面 / 側面 / 上下ビュー
- 自動回転と停止
- カメラ映像と姿勢推定を使ったAR試着（肩・腰への自動フィット）
- AR上で柄・番号・複数ロゴを別レイヤー描画し、ロゴの縦横比を維持
- 身長・胸囲・好みのフィット・推定肩幅によるサイズ提案
- PNG画像出力、プレビュー付き「マイコレクション」保存
- PC / タブレット / スマートフォン対応
- ユーザー提供GLBモデルを実際の編集対象として使用

柄、ロゴ、名前、番号は共通のUVテクスチャに合成されるため、平面レイヤーとして浮かず、ユニフォームの曲面に沿って表示されます。

前面・背面を分離した安定UV投影、襟まわりの保護帯、ロゴ縦横比補正、布目バンプと柔らかな陰影を適用しています。デスクトップではページ全体を一画面に固定し、編集パネルのみスクロールします。

左右の縫い目では背面柄を鏡合わせにして連続させ、襟は柄を残しながらベース色へ滑らかにブレンドします。ロゴは横位置に応じた曲率補正を行い、名前はサイズ変更に対応しています。初期カメラは最小表示から開始します。

AR試着のカメラ映像はブラウザ内でのみ処理し、保存・送信しません。姿勢推定には MediaPipe Pose を使用するため、AR初回起動時はインターネット接続とカメラ許可が必要です。サイズ提案は目安であり、購入前にはメーカーの実寸表との照合を推奨します。
